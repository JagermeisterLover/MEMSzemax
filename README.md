This program calculates data for pixels for MEMS surface in Zemax Opticstudio.
You can draw on 64x48 grid, or load your own image that will be converted to B/W 64x48 image. 

Guide:

1. Draw something/load image.
2. Press calculate.
3. Press copy table to clipboard, that will copy all calculated data for pixels. You can paste that in excel and copy pixel values into zemax opticstudio.

Running:

1. Install python 3.12
2. Install dependencies via pip install -r -requirements
3. python MEMS.py
